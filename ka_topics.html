Page purpose (for workflow and interface designers): This is the topic entry point for Knowledge Atlas — the page that lets any user see what intellectual territory Atlas covers before asking a question or proposing a study. Each cluster card shows the corpus size (papers already ingested), the Value of Information (VOI) score (expected reduction in uncertainty if a new well-powered study were run on that topic), and quick-action buttons routing to the Ask, Evidence, Question Maker, and Study Designer tools. Four browsing modes (All, Emerging Fronts, Most Studied, Best Next Questions) provide different entry points depending on the user's interests and goals. Clicking a card opens a detail panel showing full metadata, theories, constructs, and metrics. In the live system, these topic cards are generated from the current rebuild's live research-front data. They are overlays on the broader topic map, not the base topic ontology.
