Page purpose (for workflow and interface designers): This is the topic entry point for Knowledge Atlas — the page that lets any user (student, researcher, contributor) see what intellectual territory KA covers before they ask a question or propose a study. Each cluster card shows the corpus size (papers already ingested), the Value of Information (VOI) score (expected reduction in uncertainty if a new well-powered study were run on that topic), and quick-action buttons routing to the Ask, Evidence, Question Maker, and Hypothesis Builder tools. The priority banner surfaces the top-4 most urgent gaps so students know immediately where a new literature search would have the greatest impact. In the live system, cluster metadata (n, VOI, top constructs) is computed nightly from the finding_clusters database table; in this demo it is seeded from the CLUSTERS data array shared with the Question Maker page.
