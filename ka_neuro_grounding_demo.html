How to Use

The Neuroscientific Grounding Modal is a reusable component that can be integrated into any Knowledge Atlas page.

Integration Instructions

1. Include the CSS on any page: <link rel="stylesheet" href="ka_neuro_grounding_modal.css">

2. Include the JavaScript before closing </body>: <script src="ka_neuro_grounding_modal.js"></script>

3. Open Article Modal: window.NeuroGroundingModal.openForArticle('article-001')

4. Open Topic Modal: window.NeuroGroundingModal.openForTopic('topic-circadian')

Feature Summary

  • ✓ Article modals show all mechanistically grounded claims with mechanism chains and confidence scores
  • ✓ Identifies grounding gaps for ungrounded claims, highlighting research opportunities
  • ✓ Topic modals show aggregate grounding statistics and top mechanism chains
  • ✓ CCI confidence bars show evidence strength (strong, well-supported, plausible, speculative)
  • ✓ Bridge type pills indicate warrant type (Mechanism, Empirical Association, Functional, etc.)
  • ✓ Smooth animations and keyboard accessibility (Escape to close)
  • ✓ Fully responsive design, works on all screen sizes
