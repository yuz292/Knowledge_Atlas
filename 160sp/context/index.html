AI Context Files — COGS 160

These markdown files give your AI assistant the domain knowledge it needs to work on K-ATLAS tasks. Load the appropriate file at the start of your session before writing prompts.

How to Use These Files

Each context file below describes the K-ATLAS system from the perspective of a specific track (Image Tagger, Article Finder, VR Production, or GUI Evaluation). When you open your AI session and are assigned to a track, download and load the corresponding context file. You can do this by:

1. Downloading the .md file and reading it into your AI session prompt, or 2. Pasting the file contents directly into a chat message, or 3. Saving it locally and referencing it with: "I have a context file for this task. Let me share it with you." This ensures your AI has the precise domain knowledge, warrant structure, data models, and acceptance criteria for producing correct output.
