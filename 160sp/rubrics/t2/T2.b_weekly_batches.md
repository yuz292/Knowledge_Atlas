# T2.b — Weekly 20-article batches (three weeks, 5 pts each)

**Track**: T2 (Article Finder)
**Span**: Weeks 3–5 (three batches)
**Hardness**: Medium (per batch)
**Points**: 15 total (5 × 3 batches)
**Verification**: Automated (registry count + dedup) + AI re-grade of relevance on each batch

---

## What the student must produce

Each of three weeks:

1. **Twenty articles** submitted to the registry within the week's window, with full metadata (as for T2.a).
2. **A short batch note** (~ 150 words) per batch listing: the sub-sub-topic covered this week, one article that surprised the student, one near-miss the student almost kept.
3. **Dedup check passed**: zero duplicate DOIs across the student's own prior submissions.

## Machine-readable spec

```yaml
deliverable_id: T2.b
points: 15   # 5 per batch across 3 batches
batches:
  - label: "T2.b.1 Week 3 batch"
    span: { start: 2026-04-27, end: 2026-05-04 }
    points: 5
  - label: "T2.b.2 Week 4 batch"
    span: { start: 2026-05-04, end: 2026-05-11 }
    points: 5
  - label: "T2.b.3 Week 5 batch"
    span: { start: 2026-05-11, end: 2026-05-18 }
    points: 5
completeness_per_batch:
  required_artefacts:
    - type: registry_rows
      filter: { submitted_by: "{student_id}", submitted_at_between: "{batch.span}" }
      min_count_for_score_2: 20
      min_count_for_score_3: 20
    - type: prose_doc
      source: course_shared_doc
      pattern: "T2.b.{batch_index} — {student_name}"
      min_word_count_for_score_2: 120
      min_word_count_for_score_3: 160
    - type: dedup_check
      protocol: "Count DOI collisions between this batch and the student's prior rows; must be 0."
quality_per_batch:
  scoring_method: relevance_spot_check
  spot_check:
    n_sampled: 5
    threshold_for_score_2: 4/5
    threshold_for_score_3: 5/5
reflection_per_batch:
  scoring_method: three_sub_check
  specificity_threshold_for_score_2: 2
  specificity_threshold_for_score_3: 3
```

## Scoring bands (per batch, 5 points each)

Standard bands with dedup as a hard gate: any duplicate DOI caps Completeness at band 1.

## Late

Each batch is independent. Late on one batch does not propagate. − 1 per day late on the affected batch only.
