**Hardness**: Medium-hard
**Points**: 12
**Verification**: Automated (git log + component inspection) + AI re-grade of component API doc

## What the student must produce

1. **A second interactive scene** implementing a different T1 framework (cannot reuse T3.b's).
2. **A reusable component library** — at least three Unity prefabs/scripts extracted from T3.b that are used in T3.c without modification. Prefabs must live in `Assets/Library/` with a one-paragraph API doc per component in a `README.md`.
3. **A 250-word design log** comparing the two scenes and describing what the library abstraction was.
