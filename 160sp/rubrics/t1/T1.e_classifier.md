# T1.e — Fine-tune / select classifier + error analysis

**Track**: T1 (Image Tagging)
**Span**: Weeks 5–6
**Hardness**: Hard
**Points**: 15
**Verification**: Automated (git log + classifier output artefact) + AI re-grade of error-analysis writeup

---

## What the student must produce

1. **A classifier** — either fine-tuned or zero-shot selected from a list of off-the-shelf models — that predicts framework tags on a held-out 100-image test set.
2. **A Jupyter notebook or script** in `160sp/tracks/t1/{student_id}/classifier.ipynb` showing: data loading, train/test split, model choice with a one-paragraph justification, training code (if fine-tuning), test-set predictions, confusion matrix, per-class F1.
3. **An error-analysis writeup** (400–600 words) identifying the three most frequent confusion pairs, one hypothesis per pair for why the model confuses them, and one proposed intervention per pair.

## Scoring bands

### Completeness

- **0** — Notebook missing, or test predictions not submitted.
- **1** — Notebook present but runs-to-error, or test predictions incomplete (< 80 rows).
- **2** — All artefacts present; notebook runs end-to-end; test predictions on 100 images; error-analysis writeup 350+ words.
- **3** — All band-2 conditions plus error-analysis writeup 500+ words with specific proposed interventions.

### Quality

Driven primarily by macro-F1 on the held-out test set (see spec's `performance_thresholds`) and by the quality of the error analysis (exemplar-anchored).

- **0** — Macro-F1 < 0.30 **or** error analysis does not identify real confusion pairs from the confusion matrix.
- **1** — Macro-F1 0.30–0.49; analysis is surface-level.
- **2** — Macro-F1 0.50–0.69; analysis identifies real confusions and proposes plausible causes.
- **3** — Macro-F1 ≥ 0.70; analysis identifies real confusions, proposes plausible causes, **and** proposes concrete interventions that would move the confusion matrix in the right direction (e.g., add training data for a specific underrepresented class; revise a schema definition).

### Reflection

Standard bands. Prompt: "What did you learn about the tag schema by watching a model fail on it?"
