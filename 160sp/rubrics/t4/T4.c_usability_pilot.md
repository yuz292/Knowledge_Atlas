# T4.c — Moderated usability pilot (n = 5)

**Track**: T4
**Span**: Weeks 5–6
**Hardness**: Hard
**Points**: 15
**Verification**: Automated (data-file presence) + AI re-grade of findings + instructor audit on one session

---

## What the student must produce

1. **Five participants** recruited from outside the class (consent per class protocol).
2. **A study protocol** (~ 400 words) with the tasks, think-aloud instructions, and post-session interview questions.
3. **Five session notes** (~ 300 words each) capturing: task completions, observed friction points, participant quotes (verbatim where possible).
4. **A consolidated findings document** (~ 500 words) synthesising the five sessions: which friction points recurred, which were idiosyncratic, 3 top findings with severity.
