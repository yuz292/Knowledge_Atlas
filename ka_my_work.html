This page turns saved items into something legible. Instead of scattering notes, articles, topics, evidence, and questions across separate pages, Atlas should let you gather them into folders, review them, and export them as a working dossier.

The current workspace persists in the browser. That is already better than having no visible collection at all, but it is not yet server-backed. The next step is to sync this state through the auth server so work survives device changes.
