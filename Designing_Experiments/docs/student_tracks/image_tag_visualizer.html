🛠 Building This Page — Assignment for Pair 3

This page is currently a design prototype with placeholder image tiles. Your task is to wire it to the real image database so every tile shows an actual photograph with its computed tag values.

The data pipeline you need:

  • Query the image database: SELECT image_id, path, common_sense_label, tag_values_json FROM images WHERE json_extract(tag_values_json, '$.{tag}.whole_image') >= threshold
  • Group results by common_sense_label (from the augmented registry's common_sense_terms, matched at ingest time by VLM)
  • Sort groups by count descending; sort images within groups by tag value descending
  • Serve as JSON from a lightweight Flask endpoint — the page fetches it on load and on control change
  • The spatial breakdown table (above) loads the cells array from the tag JSON for the selected image

Success criterion: Give the page to a VR track student. Ask them: "Find me 5 images where the upper-centre region (ceiling area) scores above 0.6 on fractal dimension." They should be able to do it in under 2 minutes without any instruction from you.

# Flask endpoint sketch (backend/api/tag_browser.py)
from flask import Blueprint, jsonify, request
import sqlite3, json

bp = Blueprint("tag_browser", __name__)

@bp.route("/api/tag-browser")
def tag_browser():
    tag     = request.args.get("tag", "cnfa.fractal_dimension")
    thresh  = float(request.args.get("threshold", 0.0))
    region  = request.args.get("region", "whole_image")
    group_by= request.args.get("group_by", "common_sense")

    conn = sqlite3.connect("image_tags.db")
    rows = conn.execute("""
        SELECT image_id, thumb_path, common_sense_label, building_type,
               json_extract(tag_values, ?) AS score
        FROM   tagged_images
        WHERE  json_extract(tag_values, ?) >= ?
        ORDER  BY score DESC
    """, (f"$.{tag}.{region}", f"$.{tag}.{region}", thresh)).fetchall()

    # Group and serialize
    groups = {}
    for img_id, thumb, cs_label, btype, score in rows:
        key = cs_label if group_by == "common_sense" else btype
        groups.setdefault(key, []).append({
            "id": img_id, "thumb": thumb, "score": round(score, 3)})
    return jsonify([{"label": k, "images": v} for k,v in
                    sorted(groups.items(), key=lambda x: -len(x[1]))])
